# digit-ui-react-components


## Install

```bash
npm install --save digit-ui-react-components
```

## Usage

```jsx
import React, { Component } from "react";

import MyComponent from "digit-ui-react-components";

class Example extends Component {
  render() {
    return <MyComponent />;
  }
}
```
